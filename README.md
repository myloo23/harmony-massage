# Harmony Massage Website (Vite + React)

Egy modern, mobilbarát, konverzióra optimalizált egyoldalas weboldal a Harmony Massage (Dabas) számára.

## Fejlesztői futtatás

1. Függőségek telepítése:
   ```bash
   npm install
   ```
2. Fejlesztői szerver indítása:
   ```bash
   npm run dev
   ```
3. Production build:
   ```bash
   npm run build
   ```
